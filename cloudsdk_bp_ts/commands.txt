INITIALIZE:
----------------------------------------------------------------------------------------
npm i -g @nestjs/cli
nest new cloudsdk_bp_ts

BUILD & RUN:
----------------------------------------------------------------------------------------
npm run start:dev

ADD CUSTOM ROUTE:
----------------------------------------------------------------------------------------
nest g controller business-partner
nest g service business-partner


GENERATE SERVICE ENTITIES:
----------------------------------------------------------------------------------------
npm install -D @sap-cloud-sdk/generator@next
npm i @nestjs/config
npx generate-odata-client --input service-specifications --outputDir services


ADDITIONAL PACKAGES:
----------------------------------------------------------------------------------------
npm install @sap-cloud-sdk/http-client@next @sap-cloud-sdk/resilience@next @sap-cloud-sdk/odata-v2@next @sap-cloud-sdk/connectivity@next


DEPLOY TO CF:
----------------------------------------------------------------------------------------
cf create-service destination lite mydestination
cf create-service xsuaa application myxsuaa
cf create-service xsuaa application mysecurexsuaa -c xs-security.json

cf push

cf set-env cloudsdk_bp_ts APIKEY <YOUR APIKEY GOES HERE>
cf set-env cloudsdk_bp_ts DESTINATION_NAME bpdestination
cf set-env cloudsdk_bp_ts URL https://sandbox.api.sap.com/s4hanacloud/
