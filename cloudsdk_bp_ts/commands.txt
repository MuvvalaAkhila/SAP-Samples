INITIALIZE:
----------------------------------------------------------------------------------------
npm i -g @nestjs/cli
nest new cloudsdk_bp_ts

BUILD & RUN:
----------------------------------------------------------------------------------------
npm run start

ADD CUSTOM ROUTE:
----------------------------------------------------------------------------------------
nest g controller business-partner
nest g service business-partner


GENERATE SERVICE ENTITIES:
----------------------------------------------------------------------------------------
npm install -D @sap-cloud-sdk/generator@next
npm i @nestjs/config
npx generate-odata-client --inputDir service-specifications --outputDir services


ADDITIONAL PACKAGES:
----------------------------------------------------------------------------------------
npm install @sap-cloud-sdk/http-client@next @sap-cloud-sdk/resilience @sap-cloud-sdk/odata-v2
