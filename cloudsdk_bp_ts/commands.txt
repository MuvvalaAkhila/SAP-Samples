INITIALIZE:
----------------------------------------------------------------------------------------
npm i -g @nestjs/cli
nest new cloudsdk_bp_ts

BUILD & RUN:
----------------------------------------------------------------------------------------
npm run start:dev

ADD CUSTOM ROUTE:
----------------------------------------------------------------------------------------
nest g controller business-partner
nest g service business-partner


GENERATE SERVICE ENTITIES:
----------------------------------------------------------------------------------------
npm install -D @sap-cloud-sdk/generator
npm i @nestjs/config
npx generate-odata-client --input service-specifications --outputDir services --optionsPerService service-specifications/options-per-service.json


ADDITIONAL PACKAGES:
----------------------------------------------------------------------------------------
npm install @sap-cloud-sdk/http-client @sap-cloud-sdk/resilience @sap-cloud-sdk/odata-v2


DEPLOY TO CF:
----------------------------------------------------------------------------------------
cf create-service destination lite mydestination
cf create-service xsuaa application myxsuaa

cf push

cf set-env cloudsdk_bp_ts APIKEY <YOUR APIKEY GOES HERE>
cf set-env cloudsdk_bp_ts DESTINATION_NAME mydestination
